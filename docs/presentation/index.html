05 // Kod & CI/CD

Framtida planer

Refresh tokens

Förbättra användarupplevelsen genom refresh tokens och minska behovet av att logga in igen när JTW löper ut.

Använda poäng

Implementera funktionalitet för att kunna använda de poäng som tjänats ihop. Föräldrar ska kunna skapa "produkter".

Införa grupper

Utöka användbarheten genom att införa grupper, inte bara ha förälder-barn-relationen mellan användare.

Mobilapp (PWA)

Göra Chorely tillgänglig som en installerbar app för ännu snabbare åtkomst i farten.

Testa Spring Security

End-to-end-säkerhet: Att bara testa med mocks går förbi filterkedjan. Genom att använda en MySQL-instans med Testcontainers kan jag köra riktiga integrationstester och verifiera att säkerheten håller hela vägen.

Agentdriven utveckling

AI-utvecklad frontend: Trots noggrant arbete med att skapa överenskomna arbetsflöden frångick Mistral Vibe ofta dessa och agerade på egen hand, vilket introducerade buggar och skapade frustration när många uppgifter behövde korrigeras.
